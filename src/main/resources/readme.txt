physicalFeatures (pf) - команда, позволяющая просмотреть все имеющиеся features или обновить плагин (permission - pf.perms.pf)
physicalStats (ps) - команда, позволяющая просматривать текущие характеристики игрока (permission - pf.perms.ps)
getUpPlayer (gup) - команда, позволяющая "уронить" игрока на заданное количество секунд (permission - pf.perms.gup)
setPlayerFall (spf) - команда, позволяющая моментально "поднять" игрока если тот "упал" на землю (permission - pf.perms.spf)
